# BestB4 Marketplace — Interactive Prototype

Live, offline-capable prototype for **BestB4** — a marketplace for red-sticker / near-expiry grocery deals.
_"Good for you. Good for the planet."_

The entire app (markup, logic, images, fonts) is inlined into a single `index.html`, so it runs anywhere with no build step and no dependencies.

## View it locally
Just open `index.html` in any modern browser.

## Deploy to Vercel
1. Push this folder to a GitHub repo.
2. In Vercel, **New Project → Import** the repo.
3. Framework preset: **Other**. Leave build & output settings empty.
4. Deploy. Vercel serves `index.html` at the root automatically.

Or from the Vercel CLI, in this folder:

```
vercel
```

## Deploy to GitHub Pages
1. Push this folder to a GitHub repo.
2. **Settings → Pages → Build and deployment → Source: Deploy from a branch.**
3. Choose your branch and the root (`/`) folder. Save.
4. Your site publishes at `https://<user>.github.io/<repo>/`.
